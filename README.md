# ImmiTracker

Your complete guide to US immigration — visa information, status tracking, and green card priority date updates all in one place.

## Overview

ImmiTracker is a web application that provides detailed, up-to-date information for US immigrants and non-immigrants. Whether you are navigating student visa extensions, employer-sponsored work visas, or the path to permanent residency, ImmiTracker brings everything together in a single, easy-to-use platform.

## Features

### Visa Information
Comprehensive guides covering:
- **OPT (Optional Practical Training)** — eligibility, application process, and timelines for F-1 students
- **STEM OPT Extension** — 24-month extension requirements, employer E-Verify obligations, and reporting deadlines
- **H-1B** — cap-subject lottery, cap-exempt options, filing windows, and transfer procedures
- **H-4** — dependent visa requirements and H-4 EAD eligibility
- **PERM Labor Certification** — recruitment process, filing requirements, and common audit triggers

### Green Card Tracking (绿卡排期)
- Automatically fetches the latest **Visa Bulletin** priority dates from the US Department of State
- Displays the current **Final Action Dates** and **Dates for Filing** for all preference categories and countries
- Highlights movement trends so applicants can estimate when their priority date may become current

### Additional Resources
- Timeline comparisons across visa categories
- FAQ sections for common immigration scenarios
- Glossary of USCIS and DOS terminology
